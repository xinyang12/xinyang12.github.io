Web3.py 简单入门教程

发表于 2020-04-29 更新于 2020-05-19

简单介绍

Web3.py 是一个与以太坊交互的 Python 库。我们可以使用它来对以太坊区块链上的数据进行操作，比如转账、部署合约，调用合约，发送交易等等。类似的库还有 Web3j (Java)，Web3.js (Javascript) 等等，都是各个语言的库。

配置 Infura

我们这里使用 Infura 主要是为了与真实的主网和测试网络交互。如果本地搭建了节点或者使用 Ganache ，这一步可以跳过。

Infura 可以理解为一个接口，我们通过 Infura 可以与以太坊网络进行连接，就免去了我们自己搭建以太坊节点的麻烦。

注册登陆后，进入 dashboard

点击左上角的 ETHEREUM，再点击右边的 CREATE NEW PROJECT，起一个名字，就创建成功了

进入项目后，即可看到相应的信息，其中 PROJECT ID 和 PROJECT SECRET 是比较重要的，后面项目中需要，下面的 ENDPOINTS 可以更改网络，包括主网，测试网等等。

到这里 Infura 就配置完了，后面需要用到的时候直接把上图中的连接复制一下就行了
